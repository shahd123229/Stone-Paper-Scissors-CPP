Stone, Paper, Scissors (C++)
A classic Stone, Paper, Scissors game implemented in C++ with a focus on logic and clean CLI interaction.

Game Rules:
    Rounds: Play against the computer for a chosen number of rounds.
    Logic: The computer makes random choices to challenge the player.
    Results: Real-time tracking of player vs. computer wins and draws.

Technical Skills:
    Randomization: Used srand and rand for computer decisions.
    Structures: Organized game data using C++ Structs.
    Functions: Modular code design for easy readability.
